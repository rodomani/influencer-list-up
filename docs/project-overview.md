# プロジェクト概要

## 目的

Community Dashboardは、学生と企業が参加するコミュニティの活動を効率的に管理・可視化するためのダッシュボードアプリケーションです。

## 主要な機能

### 事務局向け
- **イベント管理**: 講義やセミナーなどのイベント登録・スケジュール管理
- **出席管理**: イベント参加者の出席状況の追跡・記録
- **メンバー管理**: 学生や企業メンバーの情報管理とグループ分け
- **フォーム管理**: アンケートや申請フォームの作成・回答収集
- **タスク管理**: イベント準備タスクの進捗管理

### コミュニティ参加者向け（学生・企業問わず）
- **スケジュール確認**: 参加イベントやプロジェクトのスケジュール管理
- **活動状況確認**: 自身の活動状況・参加実績の確認

### 企業向け
- **活動可視化**: 派遣している複数メンバーの活動状況をまとめて把握
- **活動レポート**: 派遣メンバーの参加実績・貢献度の確認（メンバーには非公開の情報も含む）

## 技術スタック

### フロントエンド
- **Firebase Hosting**: 静的サイトホスティング
- **Shadcn UI**: UIコンポーネントライブラリ

### バックエンド
- **Supabase**: バックエンドプラットフォーム
  - PostgreSQL: データベース
  - Edge Functions: サーバーレス関数
  - Storage: ファイルストレージ

## 主要なエンティティ

- **組織 (Organizations)**: 企業や団体
- **メンバー (Members)**: 学生や企業から派遣された人員
- **メンバーグループ (Member Groups)**: メンバーのグループ分け
- **イベント (Events)**: 講義、セミナー、その他のコミュニティ活動
- **フォーム (Forms)**: アンケートや情報収集フォーム
- **公式プロジェクト (Official Projects)**: コミュニティの公式プロジェクト
